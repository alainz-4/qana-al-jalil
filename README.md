# Qana al Jalil

Company of Heroes 1 custom map project.

## Working files

- `notes/` for layout sketches, balance notes, and testing notes
- `exports/` for packaged map builds and shareable zips
- `source/` for editable source assets, if needed later

## SourceTree setup

1. Open SourceTree.
2. Choose **Add Existing Local Repository**.
3. Select this folder:
   `C:\Users\Alain\OneDrive\Documents\company of heroes maps`
4. Commit changes here as the map evolves.
5. Add a remote later if you want to share with friends through GitHub, GitLab, or a private server.

## Map idea

- 2v2 urban map
- River dividing the map into two halves
- Two crossable bridges
- High ground on both sides with infantry and cannon positions
- Bridge-edge defenses that can be captured
